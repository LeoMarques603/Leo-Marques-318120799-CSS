Imagens SVG

SVG é a abreviatura de Scalable Vector Graphics que pode ser traduzido do inglês como gráficos vetoriais escaláveis. Trata-se de uma linguagem XML para descrever de forma vetorial desenhos e gráficos bidimensionais, quer de forma estática, quer dinâmica ou animada.

Foi criado pela World Wide Web Consortium, responsável pela definição de outros padrões, como o HTML e o XHTML. A criação deste formato foi baseada noutros já existentes: CSS, DOM, JPEG, PNG, SMIL e XML. É um padrão aberto desenvolvido desde 1999 por um grupo de trabalho do W3C com base nos formatos PGML, da Adobe, e VML, da Microsoft, submetidos ao W3C por essas empresas em 1998.

Vantagens:Podem ser criadas com qualquer editor de texto; podem ser pesquisadas, indexadas e compreensadas; são escaláveis; podem ser printadas em alta qualidade sem perda de resolução; podem receber zoom sem perda de resolução; puro XML.

Desvantagem: Pode ter um processamento mais lento de acordo com que aumentamos a complexidade do documento

Html

HTML é o formato padrão para criação de páginas online e aplicações de web. Em conjunto com o CSS e Javascript, eles formam as pedras principais para a World Wide Web.

Css

Cascading Style Sheets (CSS) é um simples mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web

Javascript

JavaScript é uma linguagem de programação interpretada. Foi originalmente implementada como parte dos navegadores web para que scripts pudessem ser executados do lado do cliente e interagissem com o usuário sem a necessidade deste script passar pelo servidor, controlando o navegador, realizando comunicação assíncrona e alterando o conteúdo do documento exibido.

Json

Em computação, JSON (pronúncia J-son em inglês), um acrônimo de JavaScript Object Notation, é um formato compacto, de padrão aberto independente, de troca de dados simples e rápida (parsing) entre sistemas.
